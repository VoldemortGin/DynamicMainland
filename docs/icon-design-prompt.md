# DynamicMainland Icon Design Prompt

## 废土动漫风 App Icon 提示词

### English Prompt

```
A post-apocalyptic anime-style app icon for "Dynamic Mainland". 

Center composition: a glowing holographic globe of Asia/China floating above cracked, weathered concrete ruins. The globe emits warm orange (#E07A52) and cyan (#06B6D4) light rays piercing through dust particles. 

Around the globe, multiple translucent terminal screens and code fragments orbit like satellites, each showing different colored status indicators — green, blue, purple, orange — representing AI agents.

Background: deep dark sky (#0D0D0D) with faint toxic aurora borealis in teal and amber. Silhouettes of collapsed skyscrapers and overgrown steel structures frame the edges. Fine debris and ash particles drift across the scene.

Style: detailed cel-shaded anime illustration with chromatic aberration on edges, subtle film grain texture, neon glow bleeding effects. Color palette is predominantly dark with high-contrast accent lights. Think Akira meets Ghost in the Shell meets NieR:Automata UI aesthetic.

The globe should feel like a beacon of civilization and intelligence in a ruined world — technology persisting through collapse.

Square format, 1024x1024, suitable for macOS app icon with rounded corners. No text. Clean silhouette readable at 16x16.
```

### 中文提示词

```
废土动漫风格的应用图标，名为 "Dynamic Mainland"。

构图中心：一个发光的全息地球（亚洲/中国朝向），悬浮在龟裂的风化混凝土废墟之上。地球散发暖橙色（#E07A52）和青色（#06B6D4）光线穿透尘埃粒子。

地球周围，多个半透明终端屏幕和代码碎片如卫星般环绕，每个显示不同颜色的状态指示灯——绿色、蓝色、紫色、橙色——代表不同的 AI 代理。

背景：深暗天空（#0D0D0D），远处隐约可见青色和琥珀色的有毒极光。坍塌的摩天大楼和被植物侵蚀的钢铁结构剪影框住画面边缘。细碎的废墟碎屑和灰烬粒子飘浮在场景中。

风格：精细赛璐璐着色动漫插画，边缘带色差效果，微妙的胶片颗粒纹理，霓虹光溢出效果。色调以深暗为主，搭配高对比度的强调光。参考《阿基拉》《攻壳机动队》《尼尔:自动人形》的 UI 美学融合。

地球应当给人一种文明与智慧在废墟世界中延续的灯塔感——技术在崩溃中持存。

正方形，1024x1024，适合 macOS 应用图标圆角裁切。无文字。16x16 缩小后剪影依然清晰可辨。
```
